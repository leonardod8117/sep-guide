Skills


When learning how to code in multiple languages, it is essential to learn certain skills and habits in order develop your understanding of each concept.


Planning

Before starting any project, you should organize all your ideas into an outline. This allows you to be creative because you can list all the possibilities for the design and content of your project, which are endless.

Independent Learning

You may think that what you learn from the teacher is enough, however, you should continue learning on your own beyond the classroom. In order to develop your understanding of each concept addressed in SEP, you should search for basic explanations and tutorials so that you can apply what you learned

Communication

Most projects in SEP require that the students work in pairs in order to complete it, so communication between each pair is essential. When planning their projects, students should communicate their ideas to one another and ultimately decide on the design and content of the project. Also, communicating during while working on the project is important because not doing so can create an inconvenience when you work on a coding platform such as Cloud9.
